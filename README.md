# description

X(Twitter)に画像拡大機能を追加します。

X(Twitter)のプレビュー画像をクリックすると、拡大率100%でウィンドウ全体に表示されます。  
さらに、マウスのホイールを使って最大10倍まで、ズームアップができます。

投稿画像の細部を見たい時に便利です。
